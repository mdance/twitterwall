Description
-----------

What is it?

The twitter wall module provides a wall of twitter tweets based around a particular search string.

It was inspired by:

http://fixoutlook.org

Installation
------------

 1. Extract the compressed download
 
 2. Upload the directory to your site's modules directory.
    (Don't know where to put it? sites/all/modules is a good choice.)
    
 3. Go to Admin -> Site Building -> Modules and activate the Twitter Wall module 
    
Configuration
-------------

 0. Go to Admin -> User -> Permissions and configure permissions.
    
 1. Go to Admin -> Settings -> Twitter Wall and configure any admin settings

 2. Go to Admin -> Settings -> Twitter Wall -> Twitter Wall Administration to moderate any tweets
 
 3. (Optional) Customize the css style for the twitter wall.
